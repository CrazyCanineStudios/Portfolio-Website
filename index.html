Crazy's Sausages

The game is a 3D Platformer collect-a-thon game about a sausage obsessed dog. The game was built from scratch in Unity using a Rigidbody physics system instead of the character controller system used for Benji's Ex-seal-ent Adventure. This systems allows for the character to interact with the world by carrying, pushing and pulling boxes and makes the game feel more alive.
